What is DOCTYPE ?

  • Doctype is a Document type declaration is an instruction that tells the web broswer about the versions of HTML the page is written in
  • It is placed at the very top ao an HTML document,before the html tag
  • The DOCTYPE Declaration for HTML 5 is !DOCTYPE html
  • It ensure that the web browser renders the page correctly according to the specified HTML Version.
  • An incorrect or missing DOCTYPE can cause brosers to render thye page in "quirks mode"leading to inconsistent display across different broswers.
